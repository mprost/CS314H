Tech Support Warning Marker

cmorgan@aracnet.com (Clark Morgan)
(smirk, computers)

Slashdot.org recently discussed AOL's brain dead decision to release a new browser based on IE (remember, AOL bought Netscape and funded the development of Mozilla).


Slashdot reader #1 (poses question):

"Why on earth would a non-AOL user want to use an AOL-branded version
of IE?"

Slashdot reader Donoho replies:

It's a tech support warning marker.

User: My system is slow and unresponsive and it keeps asking me if I
am a bride from Soviet Russia.

Tech: Uh, ok sir. What Operating System, web browser/version?

User: I've got Microsoft Windows ME with Internet Explorer from AOL...
[click...dial tone] Hello?
