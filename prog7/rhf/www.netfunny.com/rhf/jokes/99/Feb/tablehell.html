Table Hell

hoco@best.com (Howard Cohen)
(smirk, original, Internet)

Here is a poem I wrote to cope with my angst about problems with HTML tables.

Hope you enjoy it. :-)

Peace,

Howard Cohen 

	 Table Hell 
	    by  
	Howard Cohen 



I need a song for table hell 
To help me solve and then feel well 
About this mess of cell and row 
Whose misconstruction is my woe 


I need a song for table hell 
That finds the unclosed h-t-m-l 
A chant, a mantra, a song to cope 
Might shed some light, or so I hope 


I need a song for table hell 
To sing as I search so I don't yell 
Oh Weblint help me! Find this mote! 
But it says too much; my search is rote 


I need a song for table hell 
For now I feel it's where I fell 
Open? close? which one was pending? 
I wish this nested pain was ending... 


I need a song for table hell 
To help me find that unclosed cell 
The search, it sets my mind awhirl 
One more hour and I'm gonna hurl 


And then, I find it.  A missing cell close. 
Amid three nested tables and rows, 
With colspan, rowspan and empty cell, 
I know I'll be back to table hell
